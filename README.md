
ScalatraHello
Completely working sample Scalatra project with MongoDB (via Casbah) for future reference

# ScalatraHello #

## Build & Run ##

```sh
$ cd ScalatraHello
$ ./sbt
> container:start
> browse
```

If `browse` doesn't launch your browser, manually open [http://localhost:8080/](http://localhost:8080/) in your browser.
